# Day 3 – DOM Manipulation | Counter App

## About DOM
The Document Object Model (DOM) allows JavaScript to interact with HTML
elements dynamically. It represents the webpage as objects that can be
accessed and modified.

---

##  Concepts Used

### getElementById
Selects an element using its unique ID.

### querySelector / querySelectorAll
Selects elements using CSS selectors.

### innerText vs innerHTML
- innerText → updates text only
- innerHTML → updates HTML structure

### Events (click)
Triggers actions based on user interaction.
