#  Light / Dark Theme Toggle

##  Day 4 – DOM + CSS Integration

This project demonstrates how to integrate JavaScript DOM manipulation with CSS styling
using `classList`.

##  Topics Covered
- classList.add()
- classList.remove()
- classList.toggle()
- CSS selectors
- Dynamic styling

##  Features
- Smooth light and dark theme switching
- Clean UI
- Beginner-friendly DOM logic

##  Tech Stack
- HTML
- CSS
- JavaScript
